How does VannyBot work?

VannyBot is an entirely free project: it uses repl.it and uptimerobot to stay online permanently. In order to make sure VannyBot is as fast as possible when displaying porn, all porn is stored locally with the bot. That's right, all 20000+ of them. VannyBot also uses a randomizer algorithm to reduce the chances that you get the same image twice in a row (this algorithm is global, not guild based, hence the chances not being 0% — still, it's quite good at its job).
